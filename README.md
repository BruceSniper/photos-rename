



# 图片批量重命名工具

一个用 Go 语言编写的简单实用工具,用于将目录中的图片文件按顺序重命名为 0001.jpg、0002.jpg 等连续数字格式。

> 使用场景:
> 延时拍摄时选择了包围曝光一次拍摄多张照片，后期hdr合成后文件名发生跳跃，诸如DCS003.jpg、DCS005.jpg...会影响到序列导入达芬奇制作延时视频，本工具即可自动排序重新命名

## 功能特点

- 支持多种图片格式:jpg、jpeg、png、gif、bmp、webp
- 自动按文件名排序后重命名
- 提供预览模式,安全查看重命名计划
- 使用临时文件名避免重命名冲突
- 自动跳过已经符合格式的文件

## 使用方法

### 预览模式(推荐先使用)

```bash
go run main.go <目录路径> preview
```

预览模式会显示重命名计划,但不会实际执行重命名操作。

### 执行重命名

```bash
go run main.go <目录路径>
```

### 编译后使用

```bash
# 编译
go build -o photo-rename main.go

# 预览
./photo-rename ./photos preview

# 执行重命名
./photo-rename ./photos
```

## 使用示例

假设你的 `photos` 目录中有以下文件:
```
004.jpg
007.jpg
012.jpg
```

### 1. 先预览

```bash
go run main.go ./photos preview
```

输出:
```
=== 预览模式 ===

找到 3 个图片文件

重命名计划:
----------------------------------------
004.jpg -> 0001.jpg
007.jpg -> 0002.jpg
012.jpg -> 0003.jpg
----------------------------------------
总共需要重命名 3 个文件

这是预览模式,没有实际执行重命名操作
```

### 2. 确认无误后执行

```bash
go run main.go ./photos
```

输出:
```
=== 执行模式 ===

找到 3 个图片文件

重命名计划:
----------------------------------------
004.jpg -> 0001.jpg
007.jpg -> 0002.jpg
012.jpg -> 0003.jpg
----------------------------------------
总共需要重命名 3 个文件

开始重命名...
重命名完成!
```

## 注意事项

1. **建议先使用预览模式**查看重命名计划
2. **重要文件请先备份**,虽然程序经过测试,但操作文件系统总存在风险
3. 程序会按字母顺序排序后重命名,请确认排序结果符合预期
4. 重命名从 0001 开始,不是 0000
5. 保留原文件扩展名

## 工作原理

1. 扫描指定目录,获取所有图片文件
2. 按文件名排序
3. 生成重命名计划(从 0001 开始)
4. 先将所有文件重命名为临时文件名(避免冲突)
5. 再将临时文件名改为最终文件名

## 系统要求

- Go 1.16 或更高版本

## MacOS 使用命令

```bash
# 预览模式
go run main.go /path/to/photos preview

# 执行重命名
go run main.go /path/to/photos
```
